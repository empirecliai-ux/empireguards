# Tech Stack & Dependencies

## Core Framework
- `electron`: Main platform.
- `react`, `react-dom`: Frontend UI.
- `typescript`: For unbreakable, type-safe code.

## Terminal & Shell
- `xterm`: Terminal rendering.
- `xterm-addon-fit`: Auto-resize terminal.
- `xterm-addon-webgl`: GPU-accelerated rendering.
- `node-pty`: Pseudo-terminal for real shell access.

## File Explorer & System
- `chokidar`: Fast file watching.
- `fs-extra`: Enhanced file system operations.
- `wsl-path`: Path conversion between WSL and Windows.

## UI Components & Styling
- `tailwindcss`: Styling.
- `lucide-react`: Icons.
- `framer-motion`: Smooth transitions for panes and menus.
- `clsx`, `tailwind-merge`: Utility for class management.

## State & Data
- `zustand`: State management.
- `better-sqlite3`: Local database for commands and logs.
- `crypto-js`: Encryption for passwords.

## Utilities
- `axios`: For potential NLP/Google search integration for command summaries.
- `dotenv`: Configuration management.
- `qrcode.react`: React component for generating QR codes.
