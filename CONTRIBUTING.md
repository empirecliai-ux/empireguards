# Empire Guards Contribution Protocols

**ATTENTION:** This is a high-security repository. Unauthorized modifications are strictly prohibited.

## 1. No Direct Commits
All changes must be submitted via Pull Request. Direct commits to `main` are locked.

## 2. No Deletions
Deleting core system files, security protocols, or "Guardian" logic will result in an immediate and permanent ban from the Empire organization.

## 3. Mandatory QR Stamping
All new features or code modules MUST include a generated QR Stamp (provided by the internal Guardian utility) in their documentation or comments to verify origin.

## 4. Architectural Integrity
Do not modify the Emerald-on-Slate theme or the modular pane logic without explicit approval from the High Command.

---
*Failure to comply with these rules triggers automatic rollbacks.*
